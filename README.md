# Web Apps

### [AP-MS data for 100 autism related genes](https://phagemap.ucsf.edu/public/ASD100/)
  - Provides an interactive network and table view of protein-protein interactions of 100 genes related to autism spectrum disorder(ASD). Interactions were discovered by Affinity Purification Mass Spectrometry using over-expressed proteins in HEK293T cells. 
  - Publication: [*A foundational atlas of autism protein interactions reveals molecular convergence*](https://doi.org/10.1101/2023.12.03.569805)

### [SEC-MS data for jumbophage infected Pseudomonas](https://phagemap.ucsf.edu)
  - Publication: [*Next-generation proteomics for quantitative Jumbophage-bacteria interaction mapping*](https://doi.org/10.1038/s41467-023-40724-w)
